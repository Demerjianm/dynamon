## features

* [x] Local DynamoDB (port 8000)
* [ ] Built-in Local DynamoDB
* View
  * Table view
    * [x] Sub collection view
    * [x] Table schema view
    * [ ] Nested JSON
  * [x] JSON view
* [x] Edit record
